## POO_G1 - Mario Bros
- The project is a demake of the classic arcade game Mario Bros!

Controls:
Mario
- Left arrow Move Left
- Right arrow Move Right
- Up arrow Jump

Luigi
- A move left
- D move right
- W Jump
